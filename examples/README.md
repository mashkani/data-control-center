# Example Data

These fixtures are tiny synthetic datasets for evaluating Data Control Center
without using private files.

- `customers.csv`: customer records with a simple segment field.
- `events.jsonl`: event records in newline-delimited JSON.
- `orders.parquet`: order records stored as Parquet.

The files are intentionally small and non-sensitive. They are suitable for the
five-minute tour and backend fixture validation tests.
